# Stellaris Wiki Modding References

Local copies of key modding reference pages from the [Stellaris Wiki](https://stellaris.paradoxwikis.com/Modding).

> **Version note:** These references were fetched April 2026 and reflect wiki content verified between versions 3.1–3.7. Core scripting (effects, conditions, scopes, modifiers) is stable across versions, but **v4.0+ features** (Phoenix update zones, district rework, pop groups) may not be fully covered. For 4.0+ specifics, cross-reference against vanilla game files in `<Steam>/Stellaris/common/` or web-fetch the latest wiki page.
