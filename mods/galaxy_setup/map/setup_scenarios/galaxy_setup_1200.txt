# Galaxy Setup — 1200-star galaxy option ("Colossal")
# ====================================================
# ADDITIVE: a NEW setup_scenario (not an override of vanilla huge.txt). Adds a galaxy-size
# option above vanilla's max (Huge = 1000 stars). Purpose: more EMPTY breathing room /
# contested frontier WITHOUT more pops or empires (the cheap lag dimension — pops, not
# systems, drive late-game/MP slowdown). Verified base template = D:\Stellaris\map\setup_scenarios\huge.txt.
#
# Key choices vs huge.txt:
#   - num_stars 1000 -> 1200 (+20% systems).
#   - radius 450 -> 495 — scaled by ~sqrt(1.2) so per-star SPACING stays ~equal to Huge
#     (1200/495^2 ~= 1000/450^2). Kept <500 per the vanilla radius warning. The breathing
#     room comes from MORE systems at the SAME empire count, not from sparser stars.
#   - num_empires + all empire/FE/marauder/nomad counts: IDENTICAL to Huge (default 15).
#     Deliberately NOT raised — fixed empire count over more systems = larger frontier per
#     empire (the design goal) and no extra AI cost.
#   - colonizable_planet_odds left at 1.0 — habitable DENSITY is controlled by the in-game
#     "Habitable Worlds" slider (recommend ~0.25x) + "Guaranteed Habitable Worlds" (recommend
#     1-2) as a per-empire floor. Low ambient density + a guaranteed floor approximates the
#     desired sub-linear curve (small empires get headroom; big empires don't drown in planets).
# Everything else is copied verbatim from huge.txt; re-sync on a version bump if huge changes.
